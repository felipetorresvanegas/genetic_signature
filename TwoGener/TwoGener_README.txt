This folder contains the raw input data and R code for the TwoGener analysis.

1.Raw_data.csv: genotyps of all mothers (71) and seeds (720).

ID: Family name
OffID: 0 = mother plant; any number not equal to 0 = seed
Pop: ID of focal forest fragment
Fruit: ID of each fruit
Selfed: whether each seed was selfed or not
Hac_C7 - Hc_C126: Genotype data

2. AMOVA.rmd

R code for the TwoGener analysis.
